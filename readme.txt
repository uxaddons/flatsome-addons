== Description ==
Plugins are shared by UX teams. Plugin adds addons for flatsome themes.

UX Flatsome Addons Pro
Pro version will be released in the shortest time. Let's look forward to exciting features.

Requires use of the Flatsome theme

== Installation ==
 
This section describes how to install the plugin and get it working.
From your WordPress dashboard
=1. Visit 'Plugins > Add New'
=2. Search for UX Flatsome Addons.
=3. Activate UX Flatsome Addons from your Plugins page.

Install plugin -> Go to post timeline -> Go to flatsome UX Builder select UX Addons element and configure accordingly
